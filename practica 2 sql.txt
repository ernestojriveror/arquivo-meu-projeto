Conte quantos emprestimos foi realizado do Livro com titulo: Banco de Dados Relacionais
select count (*)
from emprestimo e
join livro l on e.codliv = l.codliv
where l.titliv = 'Banco de Dados Relacionais'

Qual data teve maior volume de reservas? Se haver empate liste todas.
select datres, count(codres) as Reserva_dia
from reserva
group by datre
order by Reserva_dia desc
limit 5
